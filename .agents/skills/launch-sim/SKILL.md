# Launch Simulation

Use this skill to launch Isaac Sim or Gazebo and inspect ROS connectivity.

## Steps

1. Check the simulation configuration and relevant launch files.
2. Launch with `ros2 launch`.
3. Inspect visualization in RViz when available.
4. Verify topic connectivity with `ros2 topic list` and targeted echo/info
   commands.
