2 種類の tool を分ける: ① 観測ストア tool（§1）＝4 tier そのものを構成する（Langfuse / audit.jsonl / metrics / rosbag2…）。② producer tool（§2）＝機能 box が信号を出す側で、tool そのものは tier ではなく、その出力が ②event / ④生 に記録される（Nav2 / Open-RMF / micro-ROS…）。4 tier の正本は 観測ログ 格納先 階層分離（05:107-122）。
