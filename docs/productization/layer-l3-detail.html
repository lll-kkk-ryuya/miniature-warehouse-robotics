5. L4 / VLA との関係 03:186-194

VLA は L3 を直接置換せず、以下のいずれかで接続する（どの場合も最後は既存安全経路に渡せる command candidate）。

  • VLA grounding report を Visual Resolver / Validator の補助情報にする。03:190
  • VLA action candidate を Safety Compiler の入力にする。03:191
  • Sim-first evaluator で fixture 化し、L3 の rule を増やす材料にする。03:192
